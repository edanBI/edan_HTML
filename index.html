Q&A

  • Three main differences between Web 1/2/3:
    1. Web 1.0 consists only from static pages. Web 2.0 gave the users dynamic content which interacts with them. Web 3.0 brought the 3D design which is being used widely in websites and services. Such as museum guides, computer games, ecommerce, geospatial contexts, etc.
    2. Web 1.0 is mostly read-only, Web 2.0 is read-write and finally Web 3.0 is portable and personal.
    3. Web 1.0 is mainly 'Company focus' meaning it was intended for commercial use. Later in Web 2.0 the web became 'Community focus' and in Web 3.0 the web became 'Individual focus'.

  • Drug side effects use case differences:
    • In Web 1.0 the user needed a specific URL of a relevat page in order to see the information about drugs side effects.
    • In Web 2.0 he (the user) can look for a specific URL of a page which will include information about the side effects of the relevant drug the user is interest in.
    • In Web 3.0 he can google the name of the drug and it side effects to receive a list of URLs which include the relevant information.
